Star System Generator for GURPS 4th Edition

Welcome!

This is a little tool to generate star systems for use in any story or Pen & Paper roleplaying game.
The source code is available on GitHub.
We use the rules from GURPS Space in the background and generate some stats that are of use in GURPS Space games.

The majority of the information is not game-specific, however, and is useful to anyone.

You can select some options about the star system, or leave it completely random.

The rules often generate hostile systems, as habitable, garden-type worlds are fairly rare. If you need to generate a system with such a world, it is preferable to force it.
Note that even a garden-type world may be very cold or hot, as long as it falls in a range that humans can live in!
If you want your star system to be located in an open cluster (which makes for close neighboring systems to explore) check the box.
You can also choose to force a specific number of stars, although only 1 to 3 are possible, with 0 representing random choice.
For the naming of the planets, you can choose between various lists of names, or use a simple numbering scheme. If you use any of the name lists, you can choose to use them as a corpus for a markov chain instead and generate "similar" names. The "depth" of the chain determines how big the patterns are that the chain looks at. Higher values make the output more similar to the input, with a too-large value simply regurgitating the input.
