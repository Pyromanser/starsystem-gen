Star System Generator for GURPS 4th. Edition

Welcome!

This is a little tool to generate star systems for use in any story or Pen \& Paper roleplaying game.
We use the rules from GURPS Space in the background and generate some stats that are of use in GURPS Space games.

The majority of the information is not game-specific, however, and is useful to anyone.

You can select some options about the star system, or leave it completely random.

Do you want to force a specific number of stars? 1-3 are possible, other values are treated as random.
